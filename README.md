# farskapsportal-ui

UI for farskapserklæring

## Komme i gang

Hent repoet fra github

```
git clone https://github.com/navikt/farskapsportal-ui.git
```

Installer nødvendige pakker:

```
yarn install
```

Start dekoratøren

```
docker-compose up -d
```

Start applikasjonen lokalt:

```
yarn start
```
